# Dayflow Backend API

Backend server for Dayflow HRMS built with Node.js and Express.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/dayflow
JWT_SECRET=your_jwt_secret_key_here
NODE_ENV=development
```

3. Create `uploads` directory for file uploads

4. Start server:
```bash
npm start
# or
npm run dev
```
