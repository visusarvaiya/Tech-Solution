# Dayflow Frontend

React frontend for Dayflow HRMS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. (Optional) Create `.env` file:
```env
REACT_APP_API_URL=http://localhost:5000/api
```

3. Start development server:
```bash
npm start
```

4. Build for production:
```bash
npm run build
```
